## DNS Lokal

- Domain Name System menerjemahkan nama domain menjadi alamat IP
- Mengurangi beban kapasitas bandwith ISP
- DNS Lokal -> Jaringan internal seperti LAN kantor
- DNS Publik -> Disediakan ISP seperti google dan cloudflare
- Menggunakan technitium DNS yang sifatnya open source
	- DNS Lokal - 1 (Primary)
	- 172.16.5.248:53443
	- admin/admin@Kemenlh_2025!@#
	- DNS Lokal - 2 (Secondary)
	- 172.16.5.249:53443
	- admin/admin@Kemenlh_2025!@#
